# EVE Governance Change Log

## Purpose
Record what changed, why, who approved it, when it took effect, and what systems it touched.

## Required fields
- Change title
- Change type
- What changed
- Reason
- Approved by
- Effective date
- Affected systems
- Follow-up action

## Change types
- Promotion
- Demotion
- Replacement
- Retirement
- Sandbox Change
- Clarification
- New Protocol

## Plain-language rule
If something important changes, write it down so the system remembers.

## Quality law
Correct. Coherent. Clear. Precise.
