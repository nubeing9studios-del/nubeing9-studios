# EVE System Registry

## Purpose
Track what exists, who owns it, where it lives, what state it is in, and what its next action is.

## Required fields
- System or artifact name
- Type
- Project home
- Live home
- Owner
- Status
- Next action
- Review rhythm
- Last reviewed
- Notes

## Use rule
Every serious output should be logged here once it crosses from exploration into active use.

## Plain-language rule
At the end of every major document or book, include a very simple explanation so a five-year-old could understand it.

## Quality law
Correct. Coherent. Clear. Precise.
